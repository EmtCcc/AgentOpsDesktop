# AI/ML 工程师 - 花影项目

## 关于花影

花影是一个跨平台桌面应用，致力于将 AI Agent 工具链统一编排成可管理的工作流。我们正在寻找一位经验丰富的 AI/ML 工程师，帮助我们构建下一代 AI 工作台的核心智能层。

## 你将做什么

- 设计并实现多模型 LLM 调度架构，支持 OpenAI / Anthropic / 本地模型热切换
- 构建 Prompt 工程体系，包括模板库、版本管理和质量评估
- 设计多 Agent 协作协议，实现任务分解、调度和结果聚合
- 构建 RAG 系统，实现上下文管理和知识检索
- 建立 AI 安全与治理机制，包括内容过滤、权限控制和审计日志

## 我们期望你

**必备条件：**
- 2年以上 LLM 应用开发经验，熟悉主流 LLM API
- 深入理解 Prompt Engineering，有 token 优化经验
- 熟悉 Agent 系统范式（ReAct / Plan-and-Execute / Multi-Agent）
- 有 LangChain / LlamaIndex 等框架使用经验
- 精通 Python 或 TypeScript

**加分项：**
- 有 RAG 系统生产化经验
- 熟悉向量数据库（Chroma / Pinecone / pgvector）
- 有模型微调和 RLHF 经验
- 有 Electron 桌面应用开发经验
- 有开源 AI 项目贡献经验

## 技术栈

- **LLM SDK：** OpenAI SDK, Anthropic SDK, LangChain.js
- **Agent 框架：** LangGraph, CrewAI, 自研框架
- **向量数据库：** Chroma / Pinecone / pgvector
- **嵌入模型：** OpenAI Embeddings / Sentence Transformers
- **监控：** LangSmith, 自建 metrics
- **语言：** TypeScript (主), Python (辅)
