# AI/ML 工程师招聘方案

## 职位概述

**职位名称：** AI/ML 工程师
**优先级：** P1
**项目：** 花影 (AgentOpsDesktop)
**汇报对象：** 技术负责人

## 项目背景

花影是一个跨平台桌面应用，目标是将 CLI Agent、编码助手和自动化工作流统一编排成可管理、可追踪、可治理的 AI 团队。AI/ML 工程师将负责核心的 LLM 集成、Prompt 工程和 Agent 系统设计，是产品智能化的关键角色。

## 核心职责

1. **LLM 集成与编排**
   - 设计并实现多模型调度架构（OpenAI / Anthropic / 本地模型）
   - 构建统一的 LLM 抽象层，支持模型热切换和降级策略
   - 优化 token 消耗、延迟和成本

2. **Prompt 工程**
   - 设计和维护系统级 Prompt 模板库
   - 实现 Prompt 版本管理和 A/B 测试框架
   - 建立 Prompt 质量评估体系（自动化 + 人工）

3. **Agent 系统架构**
   - 设计多 Agent 协作协议和通信机制
   - 实现 Agent 生命周期管理（创建、调度、监控、回收）
   - 构建 Agent 能力注册和路由系统

4. **RAG 与知识管理**
   - 设计文档索引和向量检索方案
   - 实现上下文窗口管理和记忆机制
   - 优化检索准确率和相关性

5. **AI 安全与治理**
   - 实现内容安全过滤和敏感信息检测
   - 设计 Agent 权限控制和审计日志
   - 建立 AI 输出质量监控和告警机制

## 任职要求

### 必备技能

- **LLM 应用开发**
  - 2年以上 LLM 应用开发经验
  - 熟悉主流 LLM API（OpenAI、Anthropic、Google 等）
  - 深入理解 Prompt Engineering 最佳实践
  - 有 token 优化和成本控制经验

- **Agent 系统经验**
  - 理解 ReAct、Plan-and-Execute、Multi-Agent 等范式
  - 有 LangChain / LlamaIndex / AutoGen 等框架使用经验
  - 熟悉 Function Calling 和 Tool Use 机制

- **工程能力**
  - 精通 Python 或 TypeScript
  - 熟悉向量数据库（Pinecone / Milvus / Chroma 等）
  - 有 API 设计和微服务架构经验

### 加分项

- 有 RAG 系统生产化经验
- 熟悉模型微调和 RLHF
- 有 Electron 桌面应用开发经验
- 有开源 AI 项目贡献经验
- 熟悉 CI/CD 和 MLOps 流程

## 面试流程

### 第一轮：技术筛选（30分钟）

- LLM 应用开发经验讨论
- Agent 系统设计思路
- 项目经验回顾

### 第二轮：编码测试（60分钟）

- 实现一个简单的 Agent 工具调用链
- 设计一个 Prompt 模板并优化
- 代码风格和问题解决思路评估

### 第三轮：系统设计（60分钟）

- 设计多 Agent 协作架构
- 讨论 RAG 系统方案
- 评估技术决策和权衡能力

### 第四轮：文化匹配（30分钟）

- 团队协作方式
- 学习成长意愿
- 对 AI 产品的理解和热情
