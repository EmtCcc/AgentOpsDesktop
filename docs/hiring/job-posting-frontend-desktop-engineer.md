# 前端/桌面应用工程师 - 花影项目

## 关于花影

花影是一个跨平台桌面应用，致力于将 AI Agent 工具链统一编排成可管理的工作流。我们正在寻找一位经验丰富的前端/桌面应用工程师，帮助我们构建下一代 AI 工作台。

## 你将做什么

- 设计并实现跨平台桌面应用架构（Electron/Tauri）
- 构建高质量的 React 组件库
- 与后端 API 深度集成，实现实时数据流
- 优化应用性能，确保流畅的用户体验
- 参与技术决策和架构设计

## 我们期望你

**必备条件：**
- 3年以上 TypeScript 开发经验，精通 React
- 2年以上 Electron 或 Tauri 桌面应用开发经验
- 有桌面应用性能优化经验
- 良好的代码风格和工程习惯

**加分项：**
- 有 AI/LLM 应用开发经验
- 熟悉 IPC 通信和进程间协调
- 有开源项目贡献经验
- 熟悉 CI/CD 流程

## 技术栈

- Electron/Tauri + React 18 + TypeScript 5
- Zustand/Redux Toolkit 状态管理
- Vite 构建工具
- Vitest + Playwright 测试

## 我们提供

- 有竞争力的薪资待遇
- 灵活的工作方式（远程/混合）
- 参与前沿 AI 产品的机会
- 扁平化的团队结构
- 持续学习和成长的空间

## 工作地点

远程优先，支持混合办公
