# DevOps 工程师 - 花影项目

## 关于花影

花影是一个跨平台桌面应用，致力于将 AI Agent 工具链统一编排成可管理的工作流。我们正在寻找一位经验丰富的 DevOps 工程师，帮助我们构建从开发到发布的完整工程基础设施。

## 你将做什么

- 设计和维护 CI/CD 流水线（GitHub Actions），支撑多平台（macOS / Windows / Linux）构建和发布
- 实现 Electron 应用的自动化签名、公证和灰度发布
- 容器化开发和构建环境，优化开发者体验
- 建立应用崩溃报告和构建指标监控体系
- 管理依赖安全扫描和供应链安全

## 我们期望你

**必备条件：**
- 3年以上 CI/CD 流水线设计和维护经验，精通 GitHub Actions
- 精通 Docker，有容器化开发环境经验
- 精通 Shell 脚本，熟悉 Node.js 脚本编写
- 有桌面应用或移动端发布经验，熟悉代码签名流程

**加分项：**
- 有 Electron Builder / electron-updater 配置经验
- 熟悉 macOS notarization 和 Windows Authenticode
- 有 Sentry 等崩溃报告集成经验
- 熟悉 SLSA / Sigstore 供应链安全

## 技术栈

- GitHub Actions + electron-builder + esbuild
- Docker + Kubernetes（基础）
- Sentry + GitHub Insights
- Dependabot + Snyk
- Shell + Node.js

## 我们提供

- 灵活的工作方式（远程/混合）
- 参与前沿 AI 产品的机会
- 扁平化的团队结构
- 持续学习和成长的空间

远程优先，支持混合办公
