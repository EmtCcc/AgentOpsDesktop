# 安全工程师 - 花影项目

## 关于花影

花影是一个跨平台桌面应用，致力于将 AI Agent 工具链统一编排成可管理的工作流。我们正在寻找一位经验丰富的安全工程师，帮助我们构建安全可信的 AI 工作台。

## 你将做什么

- 主导应用安全治理，推动 OWASP Top 10 合规和安全发现闭环
- 对新功能进行威胁建模，设计安全架构方案
- 建立安全开发生命周期（SDL），集成 SAST/DAST/SCA 工具到 CI/CD
- 定期进行渗透测试，模拟攻击场景并输出修复建议
- 设计安全监控和告警策略，建立应急响应流程

## 我们期望你

**必备条件：**
- 3年以上应用安全或安全工程经验
- 精通 OWASP Top 10，有实际漏洞发现和修复经验
- 熟悉 Electron/Tauri 等桌面应用安全模型
- 掌握 STRIDE 等威胁建模方法论
- 能阅读和理解 TypeScript/JavaScript 代码

**加分项：**
- 有 Electron 安全审计经验
- 熟悉 IPC 通信安全和进程隔离
- 有 AI/Agent 安全领域经验（Prompt injection、Agent 沙箱逃逸）
- 持有 OSCP、CEH、CSSLP 等安全认证
- 有开源安全工具贡献经验

## 技术栈

- Electron + React + TypeScript
- Semgrep, CodeQL, Snyk, Dependabot
- Burp Suite, OWASP ZAP
- GitHub Actions（安全扫描集成）

## 我们提供

- 参与前沿 AI 安全产品的机会
- 扁平化的团队结构
- 持续学习和成长的空间
