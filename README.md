# AgentOpsDesktop

Desktop application for AgentOps.

## Project Structure

```
src/
  main/       # Electron main process
  renderer/   # UI / renderer process
  shared/     # Shared utilities between main and renderer
assets/       # Static assets (icons, images)
docs/         # Documentation
tests/        # Test suites
```

## Getting Started

```bash
npm install
npm start
```

## Development

```bash
npm run dev    # Run with dev flags
npm run lint   # Lint source files
npm test       # Run tests
```
